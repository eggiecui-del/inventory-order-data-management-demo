# Data and Privacy Notes

This repository is a sanitized demo.

It does not include real organization data, real customer records, real phone numbers, real addresses, real order data, or real financial data.

All sample data is generated by scripts:

- `generate_mock_data.py`
- `generate_order_data.py`

Do not commit generated local files such as:

- `.xlsx`
- `.csv`
- `.env`
- log files

If private data is ever used in a separate local version, it should stay outside this public demo repository.
